## Personal use

### Backend setup

Follow tutorial in the backend repo <a href="https://github.com/nathanjukes/GoTify-Profile-Tracker">here!</a>


### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
